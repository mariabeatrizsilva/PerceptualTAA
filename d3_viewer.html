Overall Analysis
This page shows overall trends that have been aggregated over all scenes (i.e. scores per each parameter have been averaged over all scenes).
The bar to the left lets you select which scenes you'd like to have considered in the plots below.
Use the Scene Explorer tab to look at statistics for individual scenes.
% change from baseline — for each parameter value, how much the score deviates from that parameter's mean across all tested values.
Each line represents that average change from baseline averaged across selected scenes, separately per resolution.
Positive value = outperforms average; negative value = underperforms.
Toggle scenes in the sidebar to include or exclude them.
Effect size — max score − min score across all tested values of a parameter, per scene per resolution.
For example, an effect size of 36.43 for alpha weight at 50% resolution means the score varied by 36.43 points depending on which alpha weight value was used.
Larger values mean that parameter has more influence on quality at that resolution.
Select a scene and parameter from the sidebar, then explore the tabs below. Use the Series toggles to show or hide individual resolutions.
